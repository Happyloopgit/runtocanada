## Getting Started

### Prerequisites

- Flutter SDK 3.x or higher
- Dart 3.x or higher
- Xcode (for iOS development)
- Android Studio (for Android development)

### Installation

```bash
# Navigate to app directory
cd app

# Install dependencies
flutter pub get

# Run the app
flutter run
```
